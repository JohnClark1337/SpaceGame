# SpaceGame

An open-world 2D space trading and combat game built with MonoGame.

## Runtime Requirements

- Windows x64
- [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) (or .NET 8 SDK)
- OpenGL-compatible GPU drivers
- Keyboard and mouse

## Building from Source

```sh
dotnet build
```

To modify spritefonts, install the MonoGame Content Builder:

```sh
dotnet tool install --global dotnet-mgcb
```

## Controls

See the in-game Controls screen (ESC > Controls) for full keybindings.
